# zhcn-main-lab

## 项目简介

`zhcn-main-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为 HTML 页面资源的集合与维护空间。

## 目录说明

```
├── pages/          # 存放各个独立 HTML 页面
├── assets/         # 页面引用的公共资源（样式、脚本、图片等）
├── README.md       # 本文件
└── index.html      # 可选：页面入口或导航
```

- **pages/**：每个子目录或文件对应一个独立 HTML 页面，命名清晰，便于查找。
- **assets/**：包含页面通用的 CSS、JavaScript、字体、图片等资源，按类别分目录存放。

## 页面归档说明

本仓库中的 HTML 页面均为独立完整的网页，彼此之间无强关联。每个页面均可直接通过浏览器打开运行，无需额外依赖（除 assets 目录下的公共资源）。

页面可能包含简单的交互、静态内容或演示功能，但不会指向任何外部特定服务或产品。

## 维护说明

- **新增页面**：在 `pages/` 下创建新的 HTML 文件或子目录，并在 `index.html`（如有）中添加链接。
- **资源管理**：若页面需要共用资源，请统一放置在 `assets/` 下，避免重复引用。
- **提交规范**：提交信息应简洁说明变更内容（如“新增示例页面”或“更新样式库”）。
- **兼容性**：页面尽量遵循现代浏览器标准，不强制兼容过时浏览器。

## 其他

本仓库仅用于学习、演示或归档目的。欢迎浏览和参考其中的 HTML 实现，但请勿将其用于任何形式的商业推广或误导性用途。
